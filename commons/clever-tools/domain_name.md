# Manage your domains

### List your domains

If you want to list your domain, use :

    clever domain

### Add or remove domains

Using `clever domain`, you can add or remove domain names.

    # Add a domain
    clever domain add <domain-name>

    # Remove a domain
    clever domain remove <domain-name>
